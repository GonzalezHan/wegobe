## Scroll 8. Cases

파일: `src/components/phase4/Cases.jsx`

### 섹션 헤더

- `우리가 만들어낸 변화들`
- `부서 곳곳의 페인포인트는 작은 자동화 조각으로 풀렸고, 그 조각들이 모여 하나의 운영 체계를 바꾸고 있습니다.`

### 카드 제목

- `시너지지원 Weekly Issue`
- `월 결산 자동 취합 시스템`
- `비용 정산 알림 봇`
- `환율 데이터 자동 스크래핑`

### 첫 카드 핵심 문구

- 설명:
  - `약 10개의 구글독스 Weekly Issue를 자동 취합하고, 제미나이 요약을 더해 마스터 독스에 최종 분석 보고서를 생성합니다.`
- 임팩트:
  - `문서 취합과 요약 분석 자동화`
- 태그:
  - `Google Docs`
  - `Gemini`
  - `Master Docs`
- 지표:
  - `원본 문서 수 / 약 10개`
  - `최종 보고서 / 1개`

### Weekly Issue 모달 탭

- `1. GAS 코드 화면`
- `2. 코드 실행 과정`
- `3. 생성된 산출물`
- `4. DEMO Live`

### Weekly Issue 모달 내부 핵심 카피

- 코드 탭:
  - `Weekly Issue 분석 스크립트`
- 실행 탭:
  - `Execution Log`
  - `Processing Status`
- 산출물 탭:
  - `Weekly Issue Master Docs`
  - `[Key Issue Summary] / 주요 이슈 5개 / Gemini 요약`
  - `[Executive Summary] / 각사 1~2개 이슈 / Gemini 요약`
  - `[Weekly Issue] / 각사 원문 취합 / 단순 병합`
- 라이브 데모 탭:
  - `Weekly Issue Demo Recording`
  - `LIVE PLAYBACK`
